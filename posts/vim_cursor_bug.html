Vim Cursor Bug

April 25, 2016

Not sure why, but in the last month or so, Vim has started leaving these little cursor blocks around.

I can replicate the issue with the following key combination: a | right-arrow | esc.

I'm using the following plugins, all fully updated:

  • pathogen
  • ctrlp
  • supertab
  • airline
  • fugitive
  • gitgutter
  • vim-go
  • sensible
It's happening with all the machines I use, whether locally or logged in via ssh.

When/if I figure out what's going on and how to solve it, I'll update this post.
